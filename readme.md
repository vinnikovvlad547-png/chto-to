# Погодный телеграм-бот

Простой телеграм-бот для получения актуальной информации о погоде по текущей геопозиции пользователя.

## Описание проекта

Бот позволяет пользователю отправить свою геопозицию и получает в ответ прогноз погоды: температуру, ощущаемую температуру, влажность и описание погодных условий с соответствующим эмодзи.

Данные берутся из OpenWeatherMap API. Бот поддерживает русскоязычный интерфейс и удобную клавиатуру с кнопками.

## Технологии

- **Python** — язык разработки
- **pyTelegramBotAPI** — библиотека для работы с Telegram Bot API
- **Requests** — для HTTP-запросов к API погоды
- **OpenWeatherMap API** — внешний сервис для получения метеоданных

## Как запустить проект

### 1. Клонируйте репозиторий
git clone https://github.com/your-username/weather-telegram-bot.git
cd weather-telegram-bot

### 2. Получите токен для погоды и самого бота
зайдите в телеграмме в botfather и получите токен,
 а также зайдите на https://api.openweathermap.org/data/2.5/weather и получите api key
